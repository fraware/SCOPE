# Governance

SCOPE v0.1 is maintained as an open reference implementation for the Scoped Scientific Authorization Protocol.

## Decision making

- Protocol schema and policy changes require review against the design specification.
- Breaking changes are deferred until v1.0 unless required for security.

## Roles

- **Maintainers** — merge contributions, cut releases
- **Contributors** — propose changes via pull request
- **Integrators** — AKTA, PF-Core, PCS adapter authors

## Release policy

Semantic versioning applies from v0.1 onward. v1.0 will stabilize schemas and grant formats for institutional pilots.
